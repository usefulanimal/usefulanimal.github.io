股票市場贏家：實現財務自由

(2022.05.30)

一、核心觀念與心態

1. 以市場為師

  • 股票市場是最好的老師，也是最嚴厲的老師。
  • 交易是跟隨市場、順從市場，從市場中學習。
  • 交易紀錄是重要的學習工具，要經常反省、審視。
  • 市場不理會過去，只看現在。

2. 預測 vs. 修正

  • 股票交易是一連串的預測，但預測並非重點。
  • 重點是建立交易系統，並在預測錯誤時及時修正。
  • 修正就像防護網，保護交易者。
  • 不追求完美預測，但追求及時修正。

3. 心理素質

  • 股票只是籌碼、工具，不要過度關注，也不要帶有情感。
  • 了解自己，選擇並調整適合自己的交易方式。
  • 克服恐懼與貪婪，避免情緒化交易。
  • 用「機會成本」取代「絕對成本」的思維，學會汰弱留強。
  • 接受不確定性，尋求簡單答案和保證是人的天性，但要避免盲從。
  • 交易是一場心理遊戲，要有高度的自由度和自律性。
  • 看淡輸贏，停損就像呼吸一樣自然。
  • 不要害怕犯錯，但要從錯誤中學習。
  • 要有耐心和毅力，持續學習和精進。

4. 人性與股市

  • 日常生活中的慣性（例如：低價買入）不一定適用於股市（可能需要追漲）。
  • 股價上漲時，人容易產生「心理剛性」，相信會繼續上漲。
  • 股價下跌時，樂觀情緒會壓制恐懼，導致錯失停損時機。
  • 還在想上一個漏接的球, 很容易連續失誤

二、技術分析與操作

1. 技術分析基礎

  • 趨勢：
    多頭：主力線向上，K線在主力線上方，一頭比一頭高（Higher Highs），紅K比黑K多；一底比一底高（Higher Lows）。
    空頭：主力線向下，K線在主力線下方，一頭比一頭低（Lower Lows），黑K比紅K多；一底比一底低（Lower Highs）。
  • 乖離：
    股價與均線的偏離程度。
    乖離率過大時，可能出現反轉（例如：長紅、跳空）。
    乖離率是短期的獲利率指標，20%~30%可能就有人獲利了結。
  • 型態：
  • 價.量:
  • 角度:
  • MACD:
  • 籌碼: 亂中有序

2. 操作策略

  • 買入時機：
    買在趨勢形成時（是買了，但買對了）。
    大盤深跌後的第一根長紅帶量，可能是追擊訊號。
    不追求最低點買入。
  • 停損/停利：
    主動停損/換股，汰弱留強。
    混亂的盤面，先減碼，看錯再買回。
    跌破主力線（例如：40-60天均線），先減碼。
    停損不停利（讓獲利奔跑）。
    用機會成本來衡量，不要死抱著虧損的股票。
  • 資金管理：
    分批買賣，控制風險。
    即使分批買賣，每次下注的勝算和金額都可能不同。
    用最小的金額練習，繳學費，並自由調配資金。
  • 追蹤交易 >> 預測交易
  • 選強/跌勢形成才動作
  • 多頭可追漲, 空頭不追漲(勇氣)

3. 選股

  • 選擇近期強勢股，特別是收盤前放大量的攻擊股。

4. 主力思維(進階)

  • 主力出貨思維：炒到高天價，然後一路壓低。
  • 從大量出手的反覆檢討、揣摩，淬鍊簡化交易方法。
  • 過去的修正並非臆測，而是汰弱留強。
  • 人生就是. 買高價, 賣更高價. <買在安全而非最低>. 違人性 要低價, 在更低價回補

三、學習與成長

1. 學習方法

  • 建立交易系統，並大量練習。
  • 從實戰中找到適合自己的交易節奏和方法。
  • 不要怕嘗試，但要分批試單。
  • 多觀察、多思考、多做筆記，並反覆檢討。
  • 用一輩子的時間去探索，拉大格局，不要急於一時的得失。

2. 避免的錯誤

  • 過度關注消息面，忽略技術面。
  • 沒有自己的交易系統，盲目跟隨他人。
  • 不設停損，死抱著虧損的股票。
  • 頻繁交易，過度交易。
  • 不學習、不反省、不進步。
  • 常人不換股的盲區：交易成本和持股的"情緒成本"；"換股錯誤的後悔"大於"不作為的後悔"。
  • 重複做同樣的事, 卻期待不一樣的結果, 是不理性的.
  • 打破長期投資/短期投機的迷思 -> 只有交易訊號.
  • 短線是初學的必要之惡，密集集訓。
  • 交易的重點是後悔, 後悔不完, 會討苦吃. 復盤不等於後悔

四、其他

  • 股票可以讓人透過努力，提高勝率。
  • 把人生財富掌握在自己手中，尋求方法，而不是靠天分。
  • (單純 + 專注 + 堅持)
